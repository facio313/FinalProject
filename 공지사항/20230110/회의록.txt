1. 회의 내용

1) 현재 사이트의 장단점

양서연
+ 역세권
- 자료 찾기 불편
- 취업 시장에 진입하는 사람으로서 접근성이 안 좋음

박형준
+
- 추천 공고가 많이 나오는데 로그인 후에도 똑같이 나옴

허근주
+
- ui가 불편하고 잡다한 기능이 너무 많음

임채리
+ 크몽 사이트가 깔끔함, 간편 로그인 시스템
- ui가 불편하고 신입 검색 시 경력자도 같이 나옴

이수현
+
- 필터링이 제대로 안 돼서 매칭도 잘 안 되고, 카테고리가 세부적이지 않음

최경수
+ 
- ui가 불편함

정리
+ 역세권, 크몽(참고), 간편 로그인
- ui, 필터링, 매칭, 카테고리가 불편함


2) 추가 기능

선생님 : 스마트 매칭 시스템

최경수 : 입사 관리, 인사 관리, 웰컴 페이지에서 로그인 시 구직, 구인 현황

허근주 :  파이썬/안드로이드스튜디오 사용, 간편 로그인, 로그인 시 검색하면 DB에 저장 후 나중에 관련 내용 보여주기, 유료상품(직종 내 컨설팅, 동회사인 금지)

양서연 : 등록된 회사 일정 관리를 달력으로, 위젯 커스텀, 화상면접(줌API 등), 채팅(파이썬, 소켓), 입퇴사 인증 후 글 작성

임채리 : 로그인 시 직렬 선택한 거 띄어놓기, 신입/경력자 나누기, 규격화한 폼에 맞게 받기, 중소기업 신뢰도(ex 합불합 알리기)

이수현 : 중소 기업 매칭, 외주/상주/프리랜서/직무 등(크몽, 위시켓), 체크박스 DB화

박형준 : 로그인 시 회원에 맞는 정보만 뜨게끔

- 대분류 : 매칭, 입사 관리, 인사 관리, 화상면접, 회원별 웰컴 페이지
- 소분류 : 위 기능들
