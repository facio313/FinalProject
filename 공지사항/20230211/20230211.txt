2조 진행사항

-최경수
1. 완료 : 메인화면, 기본 화면 틀(타일즈), 이력서 관리
2. 진행중 : 이력서 관리 나머지 항목들 비동기화
3. 할 것 : 채용 과정 관리

- 임채리
1. 완료 : 전반적인 화면구성
              커뮤니티, 현직자인터뷰 기본 CRUD
              문의하기, 문의내역 기본 C,R
2. 진행중 : 검색, 기타(화면)
3. 할 것 : 좋아요기능, 페이징, 정렬, 투명도 등

- 박형준
1. 완료 : 로그인, 아이디, 비밀번호 찾기, 공지사항
2. 진행중 : 도움말 구현
3. 할 것 : 운영자페이지, 시스템 관리

- 윤호연
1. 완료 : 뉴스 게시판 CRUD 및 기능구현
2. 진행중 : 인재 홍보 게시판 구현
3. 할 것 : 매칭 시스템 구현, 게시판 파일 업로드

- 양서연
1. 완료 : 공고 리스트 틀, 간단한 검색, 공고상세 값만 받아옴
2. 진행중 :  공고 등록
3. 할 것 :  위의 것들 완성, 컨설팅, 취업지원게시판 구현
